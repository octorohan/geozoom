# GeoZoom

A minimal research-oriented pipeline for **true paired satellite â†’ aerial super-resolution** using a conditional GAN with **spectral consistency constraints (NDVI)**.

The project focuses on **end-to-end correctness**: dataset preparation, alignment, training, evaluation, visualization, and deployment-ready export.

## Features

- True paired super-resolution (Sentinel-2 â†’ NAIP)
- Conditional GAN (pix2pix-style)
- Residual U-Net generator + PatchGAN discriminator
- Explicit **spectral consistency loss (NDVI)**
- Perceptual loss (VGG-based)
- Robust preprocessing with geospatial resampling and SIFT fallback
- Evaluation using PSNR, SSIM, LPIPS, NDVI RMSE
- Interactive Streamlit demo for inference
- Generator exported to **ONNX** for deployment

## Assumptions & Notes

* Dataset is true paired (same geographic region, different sensors).
* NDVI is computed from Sentinel-2 NIR + Red bands.
* SIFT alignment is used only as a fallback; most pairs rely on geospatial alignment.
* Training is CPU-compatible but slow for large datasets.
* Demo is for inspection and validation, not production serving.

---

## Known Limitations

* Limited dataset size (subset of SEN2NAIP)
* Single-scale super-resolution
* No temporal consistency modeling
* No cloud or API deployment

---

## Extending
